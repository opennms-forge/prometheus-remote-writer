= OpenNMS Prometheus Remote Writer · v{revnumber}
:author: The OpenNMS Group, Inc. — Incubation project

include::_attributes.adoc[]

[.lead]
An OpenNMS Horizon plugin that pushes performance data to any
Prometheus-compatible Remote Write endpoint — Prometheus, Cortex, Grafana
Mimir, VictoriaMetrics, Thanos Receive — and surfaces OpenNMS resource
context as native Prometheus labels.

[IMPORTANT]
.Incubation project — community support only
====
This is an **incubation project** of The OpenNMS Group, Inc. Community
support is available at https://opennms.discourse.group/[opennms.discourse.group].
**No commercial support is available yet.**
====
